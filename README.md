**Welcome to Switch!**

Switch is a 2D puzzle platformer, where you complete levels by switching between 2 worlds! By pressing space, you enter the black world (from white world) and vice versa, allowing you to jump on platforms that only become solid in the correct world!

**Dimensions**

There are 2 worlds, that can be entered by pressing Space. In each world, specific platforms become solid, while others become hollow. Only blue platforms are safe, regardless of the world you are in!

White dimension:
Only Blue and Black platforms are solid

Black dimension:
Only Blue and White platforms are solid

**Levels**

Currently, only the tutorial level is available. More levels and features will be added soon, so stay updated!
If you want, you can also make levels yourself! The game is coded to read .txt files and creates tiles of 20 x 20 pixels. (Note: Follow exact format of the tutorial.txt file. Height and width does not matter. To try your own platform, change code to read your file, instead of the tutorial.txt file).

**Controls:**

Left arrow / a = go left  
Right arrow / d = go right  
Up arrow / w = jump  
Space = Switch world!
